# Interview Scheduler

Book and cancel your appointments easily with this Interview Scheduler.

## Setup

Install dependencies with `npm install`.

## Running Webpack Development Server

```sh
npm start
```

## Run with this API

https://github.com/tumiduong/scheduler-api
